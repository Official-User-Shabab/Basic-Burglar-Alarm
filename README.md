# Basic-Burglar-Alarm
Simple alarm to catch out any wrongdoings with an arduino.

---

## Components
- Arduino Uno R3
- Male to Female jumper wires
- Red LED x1
- Buzzer x1
- PIR motion sensor
- USB cable
